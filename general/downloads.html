Current source and binary distributions are available through the Apache mirrors.

Note: When downloading from a mirror, please be sure to verify that checksums and signatures are correct. To do so, use the checksum and
signature files from the main Apache site at http://www.apache.org/dist/jspwiki. Find here the KEYS file, which contains all OpenPGP keys
we use to sign releases.


Previous source and binary distributions are available through the Apache archives.


Source, javadocs and build artifacts are also available via the Apache Nexus repository, which is synced to Central repo,
under org.apache.jspwiki groupId.

If you want to build directly from the sources, please check the Get and Build Source page.
